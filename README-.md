## 简易商城系统

### 项目介绍
- 前后端分离项目
- 本项目是一个简易的商城系统，包含了用户登录、注册、商品展示、购物车、订单等功能。
- 本项目使用了SpringBoot、SpringMVC、MyBatis、MySQL、Redis、MyBatis-plus等技术。
- 本项目使用了Swagger+Knife4j作为接口文档工具。 接口文档地址：地址/doc.html
- 本项目使用了Lombok简化代码。
- 

### 业务介绍
#### 用户模块实现了以下功能
- 用户注册
- 用户登录

#### 商品模块实现了以下功能
- 商品列表
- 商品详情
- 商品分类
- 商品购买
- 商品收藏

#### 购物车模块实现了以下功能
- 添加商品到购物车
- 从购物车删除商品
- 修改购物车商品数量
